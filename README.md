# 🚀 Rishi Kant Portfolio Website

This is a responsive personal portfolio website built using **HTML, CSS, and Bootstrap 5**, based on my professional resume.

---

## 📌 Features

- Fully responsive design (mobile, tablet, desktop)
- Modern UI with Bootstrap 5
- Sections included:
  - Hero / Introduction
  - About / Summary
  - Skills / Competencies
  - Professional Experience
  - Projects Showcase
  - Certification & Education
  - Contact Section
- Clean typography and card-based layout
- Smooth navigation

---

## 🛠 Tech Stack

- HTML5  
- CSS3  
- Bootstrap 5  
- Bootstrap Icons
